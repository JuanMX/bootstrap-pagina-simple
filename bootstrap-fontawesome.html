¿Y como se integran Bootstrap y Font Awesome a un proyecto web?

Yo lo hago mediante CDN.

De manera "CONCEPTUAL", hacerlo por CDN consiste en poner URLs en el HTML: puede ser, una URL en una etiqueta link para "traerte" los estilos y/o puede ser una URL en una etiqueta script para "traerte" la librería.

El CDN también se usa para integrar otras librerías a un proyecto web, por ejemplo DataTables o sweetalert2.

Para ver un ejemplo de código sobre como integrar el CDN aqui hay uno para los estilos de Bootstrap (css) y este es otro para la librería de Bootstrap (js).
